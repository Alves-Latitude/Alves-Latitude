### 🛠️ Sobre mim

- 🔧 Trabalho com infraestrutura **bare-metal**: provisionamento de servidores, gerenciamento out-of-band (IPMI / BMC / Redfish) e suporte técnico direto com clientes
- 🖥️ Já botei a mão em **Dell PowerEdge**, **Supermicro** e **ASUS** — de KVM remoto a firmware de GPU
- 🐍 Curto automatizar as coisas com **Python** — bots de Discord, scripts, ferramentas internas
- 🌐 Web com **Django** quando dá pra sair do terminal
- 🇧🇷 Baseado em São Paulo
